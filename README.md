# 🌟 About

**NEXA AI** is an AI-powered desktop voice assistant built using **Python** and **Google Gemini**.

It listens to voice commands, performs desktop automation, answers questions using AI, and follows secure development practices with environment variables.

---

# ✨ Features

- 🎙️ Wake Word Detection
- 🧠 Google Gemini AI
- 🎤 Speech Recognition
- 🔊 Natural Text-to-Speech
- 🌐 Browser Automation
- 🕒 Time & Date
- 🔐 Secure `.env` API Key
- 🧹 Automatic Audio Cleanup
- ⚡ Clean & Modular Code

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/MayankBisht-24/Nexa-AI.git
```

Go inside the project

```bash
cd Nexa-AI
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate it

### Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python main.py
```

---

# 🔑 Environment Variables

Create a file named:

```text
.env
```

Add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```
